# Battleships game
Battleships game written in JavaFX.

It is a basic game with some extensions, computer AI is a little bit primitive

1. Combat area
  * Land (brown)
  * Sea  (blue)
2. Units
  * Tanks (light green)
  * Planes (gray, T-shaped)
  * Ships (white)
3. Sound effects
